## 함수

함수의 구조

```dart
// void의 경우는 return 값이 없는 경우 사용하는 타입
리턴타입 함수명 (매개변수) {
	// 함수 내 동작 할 코드
}
```

```dart
void main() {
  int num1 = 10;
  int num2 = 20;

  // 함수 호출
  int sum = add(num1, num2);
  print(sum);
}

// 더하기 함수 정의
int add(int a, int b) {
  return a + b;
}
```

## 분기문

```dart
// if 문
if (조건문) {
	// 참일 경우 실행 할 코드
}

// else 문
if (조건문) {
  // 참일 경우 실행 할 코드
} else {
	// 거짓일 경우 실행 할 코드
}

// else if 문
if (조건문) {
	// 조건문이 참일 경우 실행할 코드
} else if (조건문2) {
	// 조건문2가 참일 경우 실행할 코드
} else {
	// 모든 조건이 거짓일 경우 실행할 코드
}

// switch 문
switch (비교 대상) {
	case (조건식) :
		// 조건식에 맞는 실행할 코드
	case (조건식2) :
		// 조건식2에 맞는 실행할 코드
	case (조건식3) :
	...
	default :
		// 모두 맞지 안흔 경우 실행할 코드
}
```

```dart
if (10 < 20) {
   print('True');
}

bool trigger = 10 > 20;

if (trigger) {
	print('참');
} else {
	print('거짓');
}
```

## 반복문

for 문

```dart
// for 문
for (기준 변수; 조건식; 가변치) {
	// 조건식이 참일 때 반복할 코드
}

for (int i = 0; i < 10; i++) {
  print(i);
}
```

for in 문

```dart
// for in문
for (단일 변수 in 컬렉션(List/Set/Map 등)) {
	// 컬랙션 내에 요소들의 수 만큼 반복할 코드
}

List<int> indexs = [0, 1, 2, 3, 4, 5];
for (final i in indexs) {
  print(i);
}
```

while 문

```dart
// while 문
while (조건식) {
	// 조건식이 참일 경우 실행될 반복문
}

int count = 0;
while (count <= 5) {
	print(count);
	count++;
}
```

do-while 문

```dart
// do-while 문
do {
	// 선행되어 반복할 코드
} while (조건);

int count = 0;
do {
	print(count);
	count ++;
} while (count <= 5);
```

## 예외처리

프로그램적으로 의도 했거나, 의도하지 않은 상황에 대한 처리를 위해

try-catch / finally 문

```dart
void main() {
  // 예외처리
  // try-catch / finally
  int num = 10;
  try {
    print(num / 0); // Infinity 에러 발생
  } catch(error) {
    print(error);
  } finally {
    // 예외가 발생했던, 안했던 try-catch 이후 실행할 코드
    print('예외처리문이 끝났다.');
  }

  print('원래는 에러 발생으로 실행되지 않는 코드');
}
```

on 문 - 특정 상황에 대한 예외처리

```dart
void main() {
 int? num2;
 try {
    // print(num ~/ 0);
    print(num2!);
  } on UnsupportedError catch (e) {
    print('0으로 나눌 수 없습니다.');
  } on TypeError catch (e) {
    print('Null 값이 참조 되었습니다.');
  }
}
```

throw/rethrow 문

```dart
void main() {
	// throw - 예외를 만들어 던지는 문
  // rethrow - 이미 잡아놓은 예외를 밖으로 내보냄
  try {
    throw Exception('Unknown Error');
  } catch (e) {
    print(e);
    rethrow;
  }
}
```
