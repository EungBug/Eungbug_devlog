# Dart란

> Dart란 Google에서 개발한 고급 프로그래밍 언어로 객체지향 프로그래밍 언어이고 안정적이고 빠르게 어플리케이션을 만들기 위해 개발 되었다.
> 두가지 컴파일러(JIT, AOT)를 보유하고 있다는 것이 특징이고, 타입 시스템을 갖추고 있어 코드의 안정성과 가독성이 좋다.

Dart의 Compiler

- JIT Compiler(Just In Time) : 코드 실행 시점에 컴파일을 수행할 수 있는 컴파일러, 빠른 개발과 디버깅 할 때 주로 사용, Hot-Reload/Hot-Restart를 구현 할 때 활용
- AOT Compiler(Ahead Of Time) : 사전에 미리 컴파일을 수행하여 높은 실행속도를 필요로 할 때 활용되는 컴파일러, 배포하기 직전 바이너리 파일을 생성할 때 사용하는 컴파일러

## 변수와 타입

변수를 선언 할 때는 선언과 동시에 타입을 선언한다.

타입 - 기본형과 확장형 두가지로 나뉜다.

가변형 타입도 있다.

- var : 최초 한 번 부여 된 타입이 고정적으로 사용
- Dynamic : 타입이 코드 진행 중 언제든 변환 가능

Dart에서의 상수

- const : compile 시점에 상수 처리 될 경우에 활용
- final : 프로그램의 진행중에 상수 처리 될 경우에 활용

Dart에서 콘솔 찍기

```dart
print('Hello World!');
```

### 변수

```dart
// 기본형
// 참/거짓 bool
bool isTrue = true;
print(isTrue);

// 정수형 int
int num = 100;
print(num);

// 실수형 double
double num2 = 3.14;
double num3 = 3; // 정수도 할당 가능
print(num2);
print(num3);

// 문자열형 String
String str = 'Hello';
print(str);

// Null형 null
Null thisIsNull = null;
print(thisIsNull);

// 가변형
var value = 1;
value = 2;
print(value);

dynamic dynamicValue = 100;
dynamicValue = 'Hello';
print(dynamicValue);
```

## 연산자

산술 연산자

```dart
// 사칙연산 (+, -, *, /)
int num1 = 10;
int num2 = 20;
print(num1 + num2); // 30
print(num1 - num2); // -10
print(num1 * num2); // 200
print(num1 / num2); // 0.5

String str1 = 'Hello';
String str2 = str + ' World';
print(str2); // Hello World

String str3 = '$str2 World'; // 문자열의 확장, 연결 시 $ 사용
print(str3); // Hello World World;
```

비교 연산자

```dart
int num1 = 10;
int num2 = 20;

print(num1 == num2); // false
print(num1 != num2); // true

print(num1 > num2); // false
print(num1 <= num2); // true
```

논리 연산자

```dart
// &&(and), ||(or), !(not)
bool bool1 = true;
bool bool2 = false;

print(bool1 && bool2); // false
print(bool1 || bool2); // true
print(!bool1); // false
```

삼항 연산자

```dart
// 논리연산 ? 참일 경우 : 거짓일 경우
String answer = 10 < 20 ? '10이 20보다 작습니다' : '10이 20보다 큽니다';
print(answer); // 10이 20보다 작습니다
```

할당 연산자

```dart
// =, +=, -=
int num1 = 10;
num1 += 20; // num1 = num1 + 20
print(num1); // 30

int num2 =  5;
num2 *= 10;
print(num2); // 50
```

Null 처리 연산자

```dart
int num1;
print(num1); // compile error!

int? num2;
print(num2); // null
```

### Null Safety 타입

2.12버전부터 Null 값에 대한 안정성을 위해 Null Safety라는 개념이 도입 됨.

- Nullable Type : null을 허용하는 타입으로 변수 뒤에 ?을 붙여서 활용한다.
- Non-nullable Type : null을 비허용하는 타입으로 변수 뒤어 !을 붙여 활용한다.

## Class

class는 객체를 만들기 위한 template으로 데이터와 코드를 그룹화하여 코드를 관리하고 객체를 쉽게 만들어 객체지향 프로그래밍을 효과적으로 활용할 수 있다.

- 필드 : 클래스 내부에서 선언된 데이터 (변수, 상수)
- 메서드 : 클래스 내부에 선언된 기능 즉, 함수
- 생성자 : 클래스 인스턴스를 생성할 때 사용되는 코드로 생성자를 통해 매개변수를 전달하거나 초기값을 설정하는 등 작업을 할 수 있다.
